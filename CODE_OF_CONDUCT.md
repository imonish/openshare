# Code of Conduct

## Our Commitment

We as contributors and maintainers of **OpenShare** are committed to providing a welcoming, respectful, and inclusive environment for everyone.

We want this community to be a positive space for collaboration, learning, and building open technology together.

---

## Expected Behavior

Participants in this project are expected to:

- Be respectful and kind to others
- Use welcoming and inclusive language
- Accept constructive feedback gracefully
- Focus on what is best for the community and project
- Show empathy toward other contributors

---

## Unacceptable Behavior

The following behaviors are considered unacceptable:

- Harassment, discrimination, or hate speech
- Personal attacks or insults
- Trolling, abusive comments, or threatening behavior
- Publishing others‚Äô private information without consent
- Any conduct that would make others feel unsafe or unwelcome

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues and pull requests
- Discussions and community forums
- Any official communication channels related to OpenShare

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

If unacceptable behavior occurs, maintainers may take appropriate action, including:

- Warning the contributor
- Temporarily or permanently banning participation
- Closing or rejecting contributions
